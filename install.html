📋 Instrukce pro instalaci:

Metoda A - Automatická (nejjednodušší):

  1. Klikni na modré tlačítko "🚀 Automatická instalace"
  2. Pokud se Excel neotevře, zkus červené tlačítko

Metoda B - Přes prohlížeč:

  1. Klikni na červené tlačítko "⚡ Instalovat přes prohlížeč"
  2. Zkopíruj zobrazovanou URL
  3. V Excelu jdi na: Vložit → Získat doplňky → zadej URL

Metoda C - Ruční stažení:

  1. Klikni na zelené tlačítko "📁 Stáhnout manifest.xml"
  2. Otevři Excel → Vložit → Získat doplňky → Moje doplňky
  3. Klikni "Nahrát můj doplněk" a vyber stažený soubor

📋 Systémové požadavky:

  • Microsoft Excel 2016 nebo novější
  • Windows 10/11 nebo macOS 10.14+
  • Připojení k internetu
💡 Tip: Po kliknutí na "Nainstalovat" se automaticky otevře Excel s nabídkou instalace. Stačí potvrdit a add-in se objeví na kartě "Domů".
