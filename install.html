🎯 NEJSPOLEHLIVĚJŠÍ ZPŮSOB INSTALACE:

Metoda 1 - Excel Online (100% funkční):

  1. Klikni na "🌐 Otevřít Excel Online"
  2. URL se automaticky zkopíruje do schránky
  3. V Excel Online: Vložit → Doplňky → Získat doplňky
  4. Klikni "Nahrát můj doplněk" a vlož zkopírovanou URL

Metoda 2 - Desktop Excel:

  1. Klikni na "📋 Zkopírovat URL add-inu"
  2. Otevři Excel na počítači
  3. Jdi na: Vložit → Získat doplňky → "Nahrát můj doplněk"
  4. Vlož zkopírovanou URL

Metoda 3 - Ruční stažení:

  1. Klikni na "📁 Stáhnout manifest.xml"
  2. V Excelu: Vložit → Získat doplňky → Moje doplňky
  3. "Nahrát můj doplněk" a vyber stažený soubor

📋 Systémové požadavky:

  • Microsoft Excel 2016 nebo novější
  • Windows 10/11 nebo macOS 10.14+
  • Připojení k internetu
💡 Tip: Po kliknutí na "Nainstalovat" se automaticky otevře Excel s nabídkou instalace. Stačí potvrdit a add-in se objeví na kartě "Domů".
