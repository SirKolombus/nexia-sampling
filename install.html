🎯 NEJSPOLEHLIVĚJŠÍ ZPŮSOB - TRVALÁ INSTALACE:

💡 TIP: Pro trvalou instalace (aby se add-in choval jako nainstalovaný plugin):
  1. Po instalaci jdi do Soubor → Možnosti → Doplňky
  2. Nastav add-in jako "Důvěryhodný"
  3. Add-in se objeví v sekci "Moje doplňky" pro snadné odinstalování

Metoda 1 - Excel Online (100% funkční + trvalá):

  1. Klikni na "🌐 Otevřít Excel Online"
  2. URL se automaticky zkopíruje do schránky
  3. V Excel Online: Vložit → Doplňky → Získat doplňky
  4. Klikni "Nahrát můj doplněk" a vlož zkopírovanou URL
  5. ✅ Zaškrtni "Důvěřovat tomuto doplňku" pro trvalou instalaci

Metoda 2 - Desktop Excel (doporučeno pro firemní použití):

  1. Klikni na "📋 Zkopírovat URL add-inu"
  2. Otevři Excel na počítači
  3. Jdi na: Vložit → Získat doplňky → "Nahrát můj doplněk"
  4. Vlož zkopírovanou URL
  5. ✅ V nastavení označit jako "Důvěryhodný doplněk"

Metoda 3 - Ruční stažení (nejbezpečnější):

  1. Klikni na "📁 Stáhnout manifest.xml"
  2. V Excelu: Vložit → Získat doplňky → Moje doplňky
  3. "Nahrát můj doplněk" a vyber stažený soubor
  4. 🔒 Add-in se uloží jako "lokální důvěryhodný doplněk"

📋 Systémové požadavky:

  • Microsoft Excel 2016 nebo novější
  • Windows 10/11 nebo macOS 10.14+
  • Připojení k internetu
� Jak odinstalovat add-in:
Po trvalé instalaci najdeš add-in v: Soubor → Možnosti → Doplňky → Moje doplňky
Tam ho můžeš jednoduše odebrat nebo deaktivovat.
