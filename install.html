🌐 Instalovat do Excel Online 📁 Stáhnout pro Desktop Excel

🎯 ZPŮSOBY INSTALACE:

💡 TIP: Pro trvalou instalaci označte add-in jako "Důvěryhodný" při instalaci.

🌐 Metoda 1 - Excel Online (nejjednodušší):

  1. Klikni na "🌐 Instalovat do Excel Online"
  2. Automaticky se otevře Excel Online s instrukcemi
  3. Následuj pokyny v prohlížeči
  4. ✅ Zaškrtni "Důvěřovat tomuto doplňku" pro trvalou instalaci

📁 Metoda 2 - Desktop Excel (pro firemní použití):

  1. Klikni na "� Stáhnout pro Desktop Excel"
  2. Uloží se soubor manifest.xml
  3. Otevři Excel → Vložit → Získat doplňky → Moje doplňky
  4. "Nahrát můj doplněk" a vyber stažený soubor
  5. 🔒 Add-in se uloží jako "lokální důvěryhodný doplněk"

📋 Systémové požadavky:

  • Microsoft Excel 2016 nebo novější
  • Windows 10/11 nebo macOS 10.14+
  • Připojení k internetu
� Jak odinstalovat add-in:
Po trvalé instalaci najdeš add-in v: Soubor → Možnosti → Doplňky → Moje doplňky
Tam ho můžeš jednoduše odebrat nebo deaktivovat.
🛡️ Bezpečnost: Tento add-in používá pouze čtení a zápis do aktuálního sešitu. Neodesílá žádná data na server.
