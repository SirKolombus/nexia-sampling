📁 JEDINÝ SPOLEHLIVÝ ZPŮSOB INSTALACE:

💡 TIP: Pouze ruční nahrání manifestu zajistí, že add-in půjde normálně odinstalovat!

📋 POSTUP INSTALACE:

  1. Stáhni manifest: Klikni na "📁 Stáhnout manifest a nainstalovat"
  2. Otevři Excel (Desktop nebo Online)
  3. Jdi na: Vložit → Získat doplňky → Moje doplňky
  4. Klikni: "Nahrát můj doplněk"
  5. Vyber: Stažený soubor manifest.xml
  6. ✅ Potvrď instalaci - add-in se uloží jako trvalý
✅ PO INSTALACI: Add-in najdeš v sekci "Moje doplňky" kde ho můžeš:
  • 🔧 Spravovat (zapnout/vypnout)
  • 🗑️ Odinstalovat (úplně odstranit)
  • ⚙️ Konfigurovat nastavení

📋 Systémové požadavky:

  • Microsoft Excel 2016 nebo novější
  • Windows 10/11 nebo macOS 10.14+
  • Připojení k internetu
🗑️ Jak odinstalovat add-in:
DESKTOP EXCEL: Soubor → Možnosti → Doplňky → Spravovat: Doplňky Office → Jít → najdi "NEXIA - Vzorkování" → Odebrat

EXCEL ONLINE: Vložit → Doplňky → Moje doplňky → najdi "NEXIA - Vzorkování" → klikni na "..." → Odebrat

💡 Pokud se add-in neobjevuje v "Moje doplňky": Znamená to, že byl nahrán jako dočasný. Proveď instalaci znovu přes "Moje doplňky" sekci.
🛡️ Bezpečnost: Tento add-in používá pouze čtení a zápis do aktuálního sešitu. Neodesílá žádná data na server.
